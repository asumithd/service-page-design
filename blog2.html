An Integrated Product Development and Design Tool Customized for Industrial Equipment Manufacturers

With electromechanical systems getting increasingly complex and customer requirements becoming increasingly specific, industrial equipment manufacturers (IEM) are now facing a host of new design and development challenges. CAD development can get slow while building large and multifaceted assemblies. The huge scope also leads to the possibilities of problems such as collisions and interference. IEMs are facing huge difficulties in creating engineered-to-order systems, which necessitates the creation of a product design from scratch.
SOLIDWORKS® is an integrated product development solution built with high-tech tools, which help handle the challenges of building large multifaceted assemblies. At the same time, this software solution helps produce high-quality products at faster speeds. Let’s look into the most obvious design challenges faced by using traditional methods and how this software can help overcome the challenges of creating large and complex assemblies and drawings. April 23, 2021
Drawing Performance

The detailing mode and the Enhanced Graphics Performance were first added to the SOLIDWORKS 2020 version. The lightweight detailing mode works at superfast speeds. It enables a drawing to be opened in seconds, allowing the user to add the intricate details. Many improvements and upgrades in the detailing mode are added in the SOLIDWORKS 2021 version. The latest upgrade offers functions such as add whole callouts as well as Detail, Break, and Crop views.


The Enhanced Graphics Performance option helps make optimum use of graphics cards to speed up the performance. In the 2021 version, the drawing environment is also moved to the graphics card to enhance its performance and results. The improvements in the Drawing performance offered in the SOLIDWORKS 2021 version include • Select and move annotations • Zoom and pan • Dynamic highlighting • Large-section views • Saving a drawing as a .dwg file • Opening a drawing • Auto insert of center mark in section view • Option to create a detail view in a section view • Import of cosmetic threads into a view • Sorting a BOM


Quickly Resolve Lightweight Components

The SOLIDWORKS 2021 version also offers improvements in the assembly performance. To evaluate the improved performance, just open the lightweight and resolved assemblies, especially ones with parts that feature many configurations and many mates and close without saving. You will be able to immediately see the improvement in the speed and performance.


When you open the assemblies in the lightweight mode, the sub-assemblies and top-level components automatically resolve. Go to the FeatureManager® design to expand an item to view the details. The sub-assembly components are designed to remain in the lightweight mode till you expand them. It prevents the need to unnecessarily load large assemblies every time, which could take up your precious time. Also, this design allows you to access only the components you need and resolve them.

Mind Your Mates

SOLIDWORKS reduces the time and effort required to choose mates by offering different types of assembly mates that speed assembly time and help create the correct assembly relationship between components. The upgrades in the SOLIDWORKS 2021 further simplify the process of choosing the right mates. This version has an updated Property Manager in Mates, which has Standard, Advanced, Analysis, and Mechanical tabs. These options make it easy to find the right mate.


Also, while creating the mates in each Slot, you can also choose the default position. This feature saves time as you don’t need to change the location type each time you add new slot mates to the design. It also prevents you from accidently selecting the wrong constraint for a slot.

Synchronize Components

When you want to create or change a configuration within a pattern, you need to refer the original instance and cross verify the configuration pattern match. It can be a time-consuming process. SOLIDWORKS 2021 offers an option to synchronize each of the component instances created within a pattern. This feature helps you quickly make the changes without any errors. To further simplify the process and reduce the possibilities of errors, this option also offers a quick drop down menu. Select the Component Properties dialog and freeze the configuration option. So, once the configuration changes, all the pattern’s seed components will also be changed accordingly.


Tracking Interference Issues

SOLIDWORKS 2021 offers a simple and effective tool to quickly find any possible issues within your design. All the data is exported to an Excel sheet for easy verification and tracking. This tool also gives you the option to capture a thumbnail image of each interference and share it with the others in your team for their review and feedback. This kind of collaborative analysis will help you quickly identify the core issue and make the necessary changes to solve the interference.


Save the Defeatured Model as a Configuration

It is essential to remove all the unnecessary details when you want to speed up the large-assembly design process. SOLIDWORKS 2021 has just the right feature to do this for you. The Silhouette method for defeaturing allows you to create a configuration from a defeatured assembly. Easily switch between the full model and the defeatured configuration to quickly substitute the components in the defeatured model. It helps save a lot of time in the entire process cycle.


Complete your design faster

SOLIDWORKS integrated 3D product development system allows industrial equipment manufacturers to effectively collaborate with all the key stakeholders, designers, and developers. It also allows you to innovate the design process and equipment based on the requirements. This integrated software solution can help efficiently manage large assemblies in less time. So, you can get your product to the market in less time and with lesser expense.
